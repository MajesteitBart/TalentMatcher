## Getting Started

### First Login

When you first access Talent Matcher v4, you'll see the main dashboard that provides an overview of your recruitment activity.

### Key Interface Elements

1. **Navigation Bar**: Located at the top, provides access to all main sections
2. **Action Buttons**: Prominently placed for common tasks
3. **Status Badges**: Color-coded indicators for quick status identification
4. **Search and Filters**: Located on listing pages for easy data discovery

## Dashboard Overview

The dashboard serves as your command center, providing real-time insights into your recruitment pipeline.

### What You'll See

- **Candidate Statistics**: Total candidates, recent additions, and activity metrics
- **Job Overview**: Active positions, recent postings, and application counts
- **Workflow Activity**: Status of AI matching processes and recent results
- **Quick Actions**: Direct access to add candidates or create new jobs

### Navigating from the Dashboard

- **Candidates View**: Click "View All Candidates" to see the complete candidate list
- **Jobs Management**: Access all job postings from the jobs section
- **Workflow Monitoring**: Track AI matching processes in real-time

## Managing Candidates

### Viewing All Candidates

The candidates listing page provides a comprehensive view of all candidates in your system.

#### Features Available
- **Search**: Find candidates by name or email
- **Status Filtering**: Filter by application status
- **Sort Options**: Order by date, name, or status
- **Quick Actions**: View details, edit, or manage applications

#### Understanding Candidate Cards

Each candidate card displays:
- **Name and Contact Information**: Primary identification details
- **Application Count**: Number of jobs applied to
- **Status Indicators**: Current application status
- **Recent Activity**: Timeline of recent interactions

### Candidate Detail Page

The redesigned candidate detail page provides a focused, single-column view of all candidate information.

#### Main Sections

1. **Candidate Summary**
   - Profile picture with initials
   - Contact information (email, phone)
   - Application count badge
   - LinkedIn profile link (if available)

2. **Applications Section**
   - List of all job applications
   - Current status for each application
   - Application history and timeline
   - Quick actions to update status

3. **Alternative Matches** *(if available)*
   - AI-suggested job matches
   - Match scores with color coding
   - Direct links to view matched jobs

4. **Resume Content**
   - Full CV text in scrollable format
   - Clean typography for easy reading

#### Key Actions Available

- **Add Application**: Quickly apply candidate to new positions
- **Update Status**: Change application status with dropdown
- **View Job Details**: Navigate to specific job postings
- **Manage Rejections**: Trigger AI matching when rejecting candidates

### Adding New Candidates

#### Step-by-Step Process

1. Click "Add Candidate" from the candidates page or dashboard
2. **Basic Information**:
   - Full name (required)
   - Email address (required)
   - Phone number (optional)
   - LinkedIn profile URL (optional)

3. **Resume Upload**:
   - Upload CV file (PDF, DOC, DOCX)
   - System will automatically parse and extract information
   - Preview parsed data before saving

4. **Review and Save**:
   - Verify all information is correct
   - Click "Save Candidate" to add to system

#### Tips for Adding Candidates

- **Complete Information**: Provide as much detail as possible for better matching
- **Quality Resume**: Upload clear, well-formatted CVs for accurate parsing
- **LinkedIn Integration**: Include LinkedIn URLs for additional profile context

### Creating New Jobs

#### Step-by-Step Process

1. Click "New Job" from the jobs page or dashboard
2. **Basic Information**:
   - Job title (required)
   - Company selection from existing companies
   - Department and location

3. **Job Details**:
   - Experience level (junior, mid, senior, lead, executive)
   - Job type (full-time, part-time, contract, etc.)
   - Remote/hybrid/onsite preference

4. **Requirements**:
   - Required skills (add multiple skills)
   - Job description (comprehensive position details)
   - Additional qualifications or preferences

5. **Review and Post**:
   - Preview job posting
   - Set initial status (draft or active)
   - Save and publish job

#### Tips for Creating Effective Jobs

- **Clear Titles**: Use descriptive, searchable job titles
- **Detailed Descriptions**: Provide comprehensive information about the role
- **Specific Skills**: List all required and preferred skills
- **Accurate Levels**: Set appropriate experience levels for better matching

### Updating Application Status

#### From Candidate Detail Page

1. Navigate to the candidate's profile
2. Find the specific application in the Applications section
3. Click the status dropdown menu
4. Select the new status
5. Add rejection reason if changing to "rejected"

#### From Job Detail Page

1. Go to the job posting
2. Find the Applications section
3. Locate the candidate in the list
4. Use the status dropdown to update
5. System will automatically track status changes

### Rejection and Alternative Matching

When rejecting a candidate, the system automatically:

1. **Triggers AI Matching**: Finds alternative job opportunities
2. **Generates Matches**: Creates list of suitable positions
3. **Calculates Scores**: Provides match percentage for each suggestion
4. **Notifies Team**: Updates workflow status for tracking

#### Viewing Alternative Matches

- **Automatic Display**: Matches appear on candidate detail page
- **Score Breakdown**: See skills, experience, and profile scores
- **Best Match Highlight**: Top recommendation is clearly marked
- **Direct Links**: Click to view full job details

## Understanding Status Indicators

### Badge Color System

The redesigned interface uses semantic color coding throughout:

#### Success States (Green)
- **Active Jobs**: Position is open and accepting applications
- **Hired Candidates**: Successful placement achieved
- **Validated Data**: Information has been verified

#### Warning States (Yellow/Orange)
- **Under Review**: Application in evaluation phase
- **Mid-Level Experience**: Indicates intermediate seniority
- **Needs Attention**: Action may be required

#### Information States (Blue)
- **New Applications**: Recently submitted candidates
- **Best Matches**: Top AI recommendations
- **Information Updates**: New data available

#### Error/Danger States (Red)
- **Rejected Applications**: Declined candidates
- **Closed Jobs**: No longer accepting applications
- **Invalid Data**: Information requires correction

#### Neutral States (Gray)
- **Draft Jobs**: Unpublished positions
- **Unknown Status**: Information not available
- **Inactive Items**: Not currently active

### Understanding Match Scores

Alternative job matches use percentage-based scoring:

#### Score Ranges
- **80-100% (Green)**: Excellent match - strong alignment
- **60-79% (Yellow)**: Good match - potential fit
- **Below 60% (Red)**: Poor match - significant gaps

#### Score Components
- **Skills Score**: Alignment with required technical skills
- **Experience Score**: Match with experience level requirements
- **Profile Score**: Overall candidate-job compatibility

## Alternative Job Matches

### How AI Matching Works

When a candidate is rejected, the system:

1. **Analyzes Candidate Profile**: Reviews skills, experience, and qualifications
2. **Searches Open Positions**: Finds all active job postings
3. **Calculates Compatibility**: Scores each potential match
4. **Ranks Results**: Orders matches by compatibility score
5. **Presents Recommendations**: Displays top 3 matches prominently

### Interpreting Match Results

#### Best Match Badge
The highest-scoring match is highlighted with a blue "Best Match" badge, indicating the strongest compatibility.

#### Score Breakdown
Each match shows individual component scores:
- **Skills**: How well candidate skills align with job requirements
- **Experience**: Match between candidate experience and job level
- **Profile**: Overall compatibility considering multiple factors

#### Action Options
For each suggested match, you can:
- **View Full Job**: Navigate to complete job description
- **Compare Details**: Review specific alignment factors
- **Take Action**: Apply candidate or save for consideration

### Managing Match Results

#### Immediate Actions
- **Apply Candidate**: Directly apply to suggested position
- **Save Match**: Keep for future consideration
- **Dismiss Suggestion**: Remove from current recommendations

#### Long-term Benefits
- **Improved Placement**: Better candidate-job fit
- **Faster Hiring**: Reduced time-to-hire through matching
- **Data Insights**: Understanding of candidate capabilities
