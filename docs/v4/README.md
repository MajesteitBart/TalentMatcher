## Pages Captured

### Main Navigation
- **Dashboard** (`/`) - Main landing page with overview statistics
- **Candidates** (`/candidates`) - Candidate management and listing
- **Jobs** (`/jobs`) - Job postings and management
- **Workflows** (`/workflows`) - AI matching workflow monitoring
- **Settings** (`/settings`) - Application configuration

### Candidate Pages
- **Add Candidate** (`/candidates/add`) - New candidate creation form
- **Candidate Detail** (`/candidates/[id]`) - Individual candidate profile

### Job Pages
- **New Job** (`/jobs/new`) - Create new job posting
- **Job Detail** (`/jobs/[id]`) - Individual job details and applicants
- **Edit Job** (`/jobs/[id]/edit`) - Modify existing job posting
